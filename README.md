# GettingActiveApp
iOS app that gives user motivational tasks based on his/her interests.

App uses Firestore database to save tasks and user info. 

User is able to register in the app and later login with these credentials. Then it is optinal to complete a poll to set up app based on your interests. 
After poll completion tasks from categories that the user has chosen during the poll are displayed. 
In case user skipped the poll, default tasks are displayer on "All tasks" page.

After completion the task is added to the user archive that can be viewed from the profile page.
